# Better Combat Disabler Issue Tracker

Welcome! This repository is dedicated **strictly to collecting bug reports, issues, and feature requests** for the mod. The source code of the project remains private.

## 📌 How to report an issue?

You can report a bug in two convenient ways:

1. **GitHub Issues (Recommended):** 
   * Navigate to the [Issues](../../issues) tab at the top of this page.
   * Click the **New Issue** button.
   * Describe the problem in detail and attach a crash report if available.

2. **Our Discord Server:**
   * If you have a quick question or prefer not to use GitHub, visit the dedicated bug-reporting channel on our server:
   * 🔗 [Join Discord](https://discord.gg/Umn96wqaP)
